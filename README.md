# MindEnergy-Wellness-Prediction-System

## 📌 Project Overview
This project is an AI-based wellness prediction system that uses deep learning to analyze lifestyle and behavioral data.

It predicts wellness based on trained data patterns and provides results using a trained model.

---

## 📁 Files in This Repository

- **MindEnergy_Project.ipynb** → Main Google Colab notebook (data processing, training, prediction)
- **mindenergy_dataset_1000.csv** → Dataset used for training
- **MindEnergy_Report_Uma.pdf** → Generated PDF report
- **README.md** → Project documentation

---

## ⚙️ Workflow

1. Load dataset (`mindenergy_dataset_1000.csv`)
2. Process and analyze data in notebook
3. Train deep learning model
4. Generate predictions
5. Produce PDF report (`MindEnergy_Report_Uma.pdf`)

---

## 🛠️ Tools Used

- Python
- Google Colab
- Pandas
- NumPy
- TensorFlow / Keras
- Scikit-learn
- FPDF (report generation)

---

## 🚀 How to Run

1. Open `MindEnergy_Project.ipynb` in Google Colab
2. Upload dataset if needed
3. Run all cells step by step
4. View predictions and report output
